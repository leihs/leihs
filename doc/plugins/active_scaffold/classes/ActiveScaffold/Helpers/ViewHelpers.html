link_to_visibility_toggle(options = {})

Creates a javascript-based link that toggles the visibility of some element on the page. By default, it toggles the visibility of the sibling after the one it‘s nested in. You may pass custom javascript logic in options[:of] to change that, though. For example, you could say :of => ’$("my_div_id")’. You may also flag whether the other element is visible by default or not, and the initial text will adjust accordingly.

114:         # :adapter and :position are one-use rendering arguments. they should not propagate.

115:         # :sort, :sort_direction, and :page are arguments that stored in the session. they need not propagate.

116:         # and wow. no we don't want to propagate :record.

117:         # :commit is a special rails variable for form buttons

118:         blacklist = [:adapter, :position, :sort, :sort_direction, :page, :record, :commit, :_method]
119:         unless @params_for

121:           @params_for[:controller] = '/' + @params_for[:controller] unless @params_for[:controller].first(1) == '/' # for namespaced controllers

122:           @params_for.delete(:id) if @params_for[:id].nil?
123:         end
124:         @params_for.merge(options)
