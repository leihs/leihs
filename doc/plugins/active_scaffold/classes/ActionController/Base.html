The view_paths functionality in Edge Rails (Rails 2.0) doesn‘t support the idea of a fallback generic template file, such as what make ActiveScaffold work. This patch adds generic_view_paths, which are folders containing templates that may apply to all controllers.

There is one major difference with generic_view_paths, though. They should not be used unless the action has been explicitly defined in the controller. This is in contrast to how Rails will normally bypass the controller if it sees a partial.
